以下のショ－トカットキ－はビュ－ワウィンドウがアクティブになっている時のみ有効です。

ペ－ジ関連

キ－ 説明
→ ← A S 選択画像の変更
F5 ペ－ジを再読込み
F6 選択画像を再読込み
F9 作品ペ－ジをブラウザで開く
ENTER 動画・音声の再生⇔一時停止
C ペ－ジにチェックを付ける⇔はずす
DELETE ペ－ジをリストから削除

ペ－ジリスト関連

キ－ 説明
↑ ↓ W D 選択ペ－ジの変更
HOME ペ－ジリストの先頭に移動
END ペ－ジリストの末尾に移動
[Ctrl+Shift]+C チェックの付いていない全てのペ－ジをリストから削除
[Ctrl+Shift]+DELETE 全てのペ－ジをリストから削除

ビュ－ワ関連

キ－ 説明
SPACE Seek ⇔ View モ－ド切り替え
TAB Icon ⇔ Seek/View モ－ド切り替え
H ヒントバ－を表示⇔非表示
L ペ－ジ選択パネルの左右の位置を切り替え
U ペ－ジ選択パネルを表示⇔非表示

ウィンドウ関連

キ－ 説明
F1 ビュ－ワウィンドウを非表示
F3 オプションウィンドウを表示⇔非表示
F4 システム情報ウィンドウを表示⇔非表示

キ－ 説明

[Ctrl]+F11 不使用メモリの開放
[Ctrl]+[Alt]+F5 現在進行中のダウンロ－ドを全てキャンセル
