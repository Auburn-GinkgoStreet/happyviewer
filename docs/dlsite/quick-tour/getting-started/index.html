準備

まず、お好みのブラウザでDLsiteの作品リストペ－ジを開きます。

作品リストペ－ジの一例

  • https://www.dlsite.com/home/
  • https://www.dlsite.com/home/ranking?date=30d

推奨ブラウザ

  • Chrome
  • FireFox
  • Safari
  • Edge

非推奨ブラウザ

  • DLsite Nest (理由: リンクのドラッグができない)
  • InternetExplorer (理由: リンクのドラッグができない)

次にHappyDLsite.exeをダブルクリックしてアプリケ－ションを起動します。
